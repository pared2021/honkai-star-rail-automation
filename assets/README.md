# 资源文件目录

此目录包含应用程序所需的各种资源文件。

## 目录结构

```
assets/
├── README.md           # 本文件
├── icon.ico           # 应用程序图标
├── templates/         # 模板图片目录
│   ├── ui_elements/   # UI元素模板
│   ├── buttons/       # 按钮模板
│   ├── menus/         # 菜单模板
│   └── dialogs/       # 对话框模板
├── sounds/            # 音效文件（可选）
└── fonts/             # 字体文件（可选）
```

## 模板图片说明

模板图片用于游戏界面元素的识别和定位。所有模板图片应该：

1. **格式要求**：PNG格式，支持透明度
2. **尺寸要求**：尽量小而精确，包含关键特征
3. **命名规范**：使用描述性名称，如 `start_button.png`、`main_menu.png`
4. **分辨率**：基于1920x1080分辨率截取

## 常用模板类型

### UI元素模板
- 开始按钮
- 设置按钮
- 确认按钮
- 取消按钮
- 关闭按钮

### 游戏界面模板
- 主菜单界面
- 战斗界面
- 背包界面
- 任务界面
- 商店界面

### 状态指示器
- 加载中图标
- 完成标识
- 错误提示
- 网络状态

## 添加新模板

1. 使用游戏内截图工具或第三方截图软件
2. 截取包含目标元素的最小区域
3. 保存为PNG格式
4. 放置在相应的子目录中
5. 使用描述性文件名

## 模板质量要求

- **清晰度**：图像清晰，无模糊
- **对比度**：与背景有足够对比度
- **稳定性**：在不同游戏状态下保持一致
- **唯一性**：避免与其他元素混淆

## 注意事项

1. 模板图片的版权属于原游戏开发商
2. 仅用于个人学习和研究目的
3. 不得用于商业用途
4. 定期更新模板以适应游戏版本变化
